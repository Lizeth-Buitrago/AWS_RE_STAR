## Código

python

def is_prime(num):
    if num < 2:
        return False
    for i in range(2, int(num ** 0.5) + 1):
        if num % i == 0:
            return False
    return True

with open("results.txt", "w") as file:
    for number in range(1, 251):
        if is_prime(number):
            file.write(str(number) + "\n")

print("Prime numbers saved to results.txt")

# Posible Error: `Permission denied`

Al escribir:

Eso intenta **ejecutar el archivo como programa**, pero tu script es de **Python**, por eso aparece:

La forma correcta de ejecutarlo es:

# Posible error 


Ocurre porque tu repositorio local está en la rama **`master`**, pero intentaste subir a **`main`**.

En tu pantalla se ve esto:


[master (root-commit) ...]


Eso significa que la rama actual es **master**, por eso aparece el error:


error: src refspec main does not match any


---

# Solución rápida

Ejecuta este comando para cambiar el nombre de la rama a **main**:


git branch -M main


Luego vuelve a subir el código:


git push -u origin main
